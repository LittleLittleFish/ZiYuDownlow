# 上线清单

## 1. 业务与合规
- 完成免责声明、用户协议、隐私政策页面
- 完成卖家入驻规则和内容审核标准
- 确认平台不保存提取码的流程文案
- 明确退款 SLA 与提现 SLA

## 2. 技术准备
- 生产环境环境变量配置完成
- 数据库迁移可执行
- Redis、PostgreSQL 可访问
- HTTPS 证书部署完成
- 域名解析完成
- CDN 和安全策略生效

## 3. 支付与财务
- 支付沙箱与生产密钥隔离
- 回调地址可访问并可验签
- 分佣比例配置正确
- 提现审核与打款流程确认
- 对账脚本跑通

## 4. 风控与监控
- 登录与支付接口限流生效
- Sentry 错误告警可用
- 关键任务失败告警可用
- 数据库备份与恢复演练完成
- 敏感操作审计日志可查询

## 5. 内容与运营
- 首页文案与引导位确认
- 官方资源首批内容准备完成
- 卖家招募页上线
- 客服话术、审核 SOP、退款 SOP 完成
- 数据日报模板可用

## 6. 验收场景
- 卖家上传资源 -> 审核通过 -> 上架
- 买家下单 -> 支付成功 -> 联系卖家 -> 确认收货 -> 分佣
- 买家超时未收到密码 -> 发起退款 -> 管理员处理
- 卖家申请提现 -> 管理员审核 -> 登记打款
